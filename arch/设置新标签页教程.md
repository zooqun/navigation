# 导航页面设为浏览器新标签页教程

## 方法一：设置为主页（推荐）

### Chrome 浏览器
1. 打开 Chrome 浏览器
2. 点击右上角三点菜单 → 设置
3. 在"启动时"部分选择"打开特定网页"
4. 点击"添加新网页"
5. 输入：`http://localhost:8000/navigation.html`
6. 点击"添加"

### Firefox 浏览器
1. 打开 Firefox
2. 点击右上角三横菜单 → 设置
3. 在"常规"选项卡的"启动"部分
4. 选择"显示我的主页"
5. 在"主页"输入框中输入：`http://localhost:8000/navigation.html`

### Edge 浏览器
1. 打开 Edge 浏览器
2. 点击右上角三点菜单 → 设置
3. 在"开始"选项中
4. 选择"打开特定网页"
5. 点击"添加新网页"
6. 输入：`http://localhost:8000/navigation.html`
7. 点击"保存"

## 方法二：直接修改浏览器配置文件

### Chrome 配置文件
文件位置：`%LOCALAPPDATA%\Google\Chrome\User Data\Default\Preferences`
需要修改 `network` 部分的 `network_prediction_enabled` 和相关设置

### Firefox 配置文件
文件位置：`%APPDATA%\Mozilla\Firefox\Profiles\[随机字符].default-release\prefs.js`
需要添加：`user_pref("browser.startup.homepage", "http://localhost:8000/navigation.html");`

## 方法三：使用浏览器扩展

可以创建一个简单的浏览器扩展，强制新标签页显示指定内容。

## 方法四：启动参数

创建浏览器快捷方式，添加启动参数：
```
"C:\Program Files\Google\Chrome\Application\chrome.exe" --new-tab http://localhost:8000/navigation.html
```

## 重要提醒

1. **保持服务器运行**：确保 `python -m http.server 8000` 始终在运行
2. **固定端口**：不要关闭运行服务器的终端窗口
3. **开机自启**：可以将服务器脚本设置为开机自启动
4. **防火墙设置**：确保防火墙允许8000端口的访问

## 一键启动脚本

创建 `start_navigation.bat` 文件：
```batch
@echo off
echo 启动导航页面服务器...
cd "c:\Users\lenovo\Documents\trae_projects\NavigationPage"
start "导航服务器" cmd /k python -m http.server 8000
start "" "http://localhost:8000/navigation.html"
echo 服务器已启动，浏览器将自动打开...
pause
```
